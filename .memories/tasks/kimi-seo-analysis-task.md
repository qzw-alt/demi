# Kimi SEO深度分析任务

## 背景

我们是一个面向海外用户的"来中国就医"中介服务网站 chinahospitalsguide.com。

Google Search Console 显示：3个月内1,190次展示，但只有1次点击。排名多数在5-9（在第一页边缘），但用户不点击。

## 已知数据

### 唯一有点击的页面
- /blog/dental-implants-china.html — 603展示，1点击，排名6.8，CTR 0.17%

### 有大量展示但零点击的页面
- / — 257展示，排名8.83
- /news/2026-03-24-japan-stem-cell-diabetes-therapy-approval.html — 215展示，排名5.53
- /news/2026-03-27-hello-china-2026-tcm-wellness.html — 109展示，排名4.97
- /stories.html — 5展示，排名3.8

### 主要目标用户
- 想来中国做医疗的外国人（欧美、东南亚、中东为主）
- 关键词方向：medical tourism China, dental implants China, IVF China, cancer treatment China

## 任务

1. **分析为什么 dental-implants 页面有点击**（0.17% CTR）——找出成功因素

2. **分析首页和新闻页面为什么零点击**（排名5-9但CTR=0%）——给出meta description优化方案

3. **关键词缺口分析**：
   - 搜索 "medical tourism China"、"dental tourism China"、"IVF treatment China" 等关键词
   - 看看搜索结果里我们有没有出现、排名第几
   - 找出我们还没有覆盖的高搜索量长尾关键词

4. **竞品对比**：
   - 找出3-5个同类竞品网站（如 Thailand medical tourism 相关的英文网站）
   - 对比他们的页面 title/metadata 写法和我们有什么不同

## 输出要求

1. 针对每个零点击页面给出具体的新 meta title 和 meta description 建议
2. 列出我们缺失的10-20个高价值关键词
3. 竞品网站的3个值得借鉴的写法特点
4. 优先级排序：先做什么有最大影响
