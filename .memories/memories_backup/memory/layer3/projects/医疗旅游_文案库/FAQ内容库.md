## 💰 费用与支付

### Q1: 为什么中国医疗费用这么低？质量有保障吗？

**简短回答:**
成本低主要源于运营成本和规模效应，而非质量妥协。中国顶级医院使用进口设备，医生经验丰富，JCI认证医院达到国际标准。

**详细解答:**
中国医疗费用较低主要有以下几个原因：

1. **运营成本差异**: 医生薪资、医院租金、人力成本相对较低
2. **规模效应**: 中国医院手术量大，经验丰富，标准化流程降低成本
3. **供应链本地化**: 医疗设备和药品采购成本较低
4. **政府支持**: 医疗基础设施投入大，鼓励医疗旅游发展

**质量保证:**
- 中国顶级医院普遍使用进口医疗设备(GE、Siemens、Philips等)
- 外科医生年手术量通常是西方同行的3-5倍
- 40+家医院获得JCI国际认证
- 多家医院与哈佛、约翰霍普金斯等建立合作关系

## ✈️ 签证与旅行

### Q11: 如何申请中国医疗签证？

**回答:**
申请中国医疗签证(M字签证)流程如下：

**所需材料:**
1. 护照(有效期6个月以上)
2. 签证申请表
3. 医院邀请函(我们协助提供)
4. 医院预约确认函
5. 资金证明
6. 往返机票预订
7. 酒店预订

**申请流程:**
1. 联系医院获取邀请函(我们协助)
2. 在线填写签证申请表
3. 预约签证中心递交材料
4. 缴纳签证费($140-200)
5. 等待4-7个工作日
6. 领取签证

**我们的服务:**
- 协助准备邀请函
- 提供签证申请指导
- 推荐签证代办服务(如需要)

---

### Q12: 医疗签证有效期多长？可以停留多久？

**回答:**

**签证类型:**
- **M字签证(医疗)**: 专为医疗目的设计
- 有效期: 3个月至1年(根据治疗计划)
- 停留期: 30-180天(可延期)

**延期:**
- 如需延长治疗，可在医院国际部申请签证延期
- 通常可延长30-90天
- 延期费用约$50-100

**多次入境:**
- 如需多次往返，可申请多次入境签证
- 适合需要多次复查的患者

## 🏃 手术与康复

### Q14: 术前需要做哪些准备？

**回答:**

**医疗准备:**
1. **国内检查**: 提供近期检查报告(X光、MRI、血液检查等)
2. **病史资料**: 整理既往病史、用药记录、过敏史
3. **视频会诊**: 与主刀医生远程沟通(可选)
4. **停药指导**: 按医嘱停用某些药物(如抗凝药)

**旅行准备:**
1. **签证**: 提前1-2个月申请
2. **机票**: 建议购买可改签机票
3. **住宿**: 预订医院附近酒店
4. **保险**: 购买医疗旅游保险

**物品准备:**
- 护照、签证、医疗文件
- 常用药物(带处方)
- 舒适的衣物和鞋子
- 个人护理用品
- 电子设备及充电器
