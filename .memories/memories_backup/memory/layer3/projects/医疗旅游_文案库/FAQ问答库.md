## 一、费用相关问题

### Q1: 为什么中国医疗费用这么低？

**简短回答:**
中国医疗费用低主要因为医疗体系不同、运营成本较低，但质量并不打折——顶级医院设备先进，医生经验丰富。

**详细回答:**
1. **医疗体系差异:** 中国政府对公立医院有大量补贴，医院运营成本较低
2. **人工和药品成本:** 中国的人工成本和药品采购成本低于欧美
3. **规模效应:** 中国顶级医院年手术量巨大，单位成本更低
4. **去除中间环节:** 医疗旅游套餐直接对接医院，没有中间商加价

**重要强调:**
低价不等于低质。中国顶级三甲医院:
- 配备最新医疗设备 (达芬奇机器人、3.0T MRI、PET-CT等)
- 医生年手术量是欧美同行的3-5倍
- 许多医生有海外培训经历
- 部分医院获得JCI国际认证

## 二、医疗质量问题

### Q4: 中国医生的资质如何？

**我们的标准:**
只合作三甲医院的副主任医师及以上职称医生。

**医生资质:**
- 职称: 副主任医师、主任医师
- 经验: 平均从业20年以上
- 手术量: 年主刀手术量超1000例
- 培训背景: 多数有海外培训经历

**举例:**
- 北京阜外医院心脏科医生: 年手术量超5000例
- 北京积水潭医院骨科医生: 年关节置换超2000例
- 对比美国顶级医院: 年手术量通常1000-2000例
