## 🏥 关于医疗质量

### Q1: 中国医院的医疗水平如何？

**关键点:**
- 北京协和、华西医院等位列全球Top 100
- 医生多有海外培训背景
- 三甲医院 = 中国最高医院等级
- JCI认证 = 国际医疗质量金标准

## 💰 关于费用

### Q5: 为什么费用比美国低这么多？
**A:** 费用差异主要来自以下因素：

1. **人力成本**: 中国医护人员薪资水平相对较低
2. **药品耗材**: 集中采购降低药品和耗材价格
3. **运营成本**: 医院管理和运营 overhead 较低
4. **保险体系**: 没有美国高额的医疗保险 overhead
5. **政府补贴**: 公立医院享受一定政府支持

**重要说明**: 费用低不代表质量差。中国顶级医院的医疗质量达到国际标准，费用优势来自结构性成本差异，而非质量妥协。

## ✈️ 关于签证和行程

### Q12: 如何申请中国医疗签证？
**A:** 

**申请流程:**
1. 我们提供医院邀请函
2. 提供医院预约确认函
3. 向当地中国领事馆申请M字签证（医疗签证）
4. 通常3-5个工作日出签

**所需材料:**
- 护照（有效期6个月以上）
- 签证申请表
- 医院邀请函
- 预约确认函
- 照片
- 签证费

---

### Q13: 需要在中国停留多久？
**A:** 根据手术类型：

| 手术类型 | 住院时间 | 建议总停留 |
|----------|----------|------------|
| 微创手术 | 1-3天 | 5-7天 |
| 关节置换 | 7-14天 | 2-3周 |
| 心脏手术 | 7-10天 | 3-4周 |
| 肿瘤手术 | 5-10天 | 2-3周 |
| 眼科手术 | 门诊 | 3-5天 |

**建议**: 术后初期康复在中国完成，便于医生观察和处理突发情况。

---

### Q14: 家属可以陪同吗？
**A:** 
- ✅ 家属可以申请陪同签证
- ✅ 医院允许家属陪护（部分医院有探视时间限制）
- ✅ 可协助安排家属住宿
- ⚠️ 家属费用需自理

---

### Q15: 术后可以立即回国吗？
**A:** 
- **短途飞行** (≤3小时): 术后1-2周可飞行
- **长途飞行** (>3小时): 建议术后3-4周
- **心脏手术**: 需医生评估，通常需4-6周
- **需要**: 医院出具适航证明

## 📋 关于治疗流程

### Q19: 治疗流程是怎样的？
**A:** 

**术前阶段:**
1. 远程咨询和病历评估
2. 制定治疗方案和费用预算
3. 签署服务协议
4. 申请医疗签证
5. 预订行程和住宿

**治疗阶段:**
1. 机场接机
2. 入院检查
3. 手术治疗
4. 术后恢复
5. 出院评估

**术后阶段:**
1. 康复指导
2. 定期随访
3. 病历邮寄
4. 长期跟踪
