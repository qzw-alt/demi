# 🔑 LanceDB-Pro 支持的嵌入模型

> 任何 OpenAI-Compatible API 都可以使用

---

## ✅ 官方支持的模型

### 1. Jina (推荐 ⭐)
| 项目 | 配置 |
|------|------|
| **模型** | jina-embeddings-v5-text-small |
| **Base URL** | https://api.jina.ai/v1 |
| **维度** | 1024 |
| **费用** | 免费额度 1M tokens |
| **特点** | 中文优化好，免费额度足 |

### 2. OpenAI
| 项目 | 配置 |
|------|------|
| **模型** | text-embedding-3-small |
| **Base URL** | https://api.openai.com/v1 |
| **维度** | 1536 |
| **费用** | $0.0001/1000 tokens |
| **特点** | 效果最好，需要付费 |

### 3. Google Gemini
| 项目 | 配置 |
|------|------|
| **模型** | gemini-embedding-001 |
| **Base URL** | https://generativelanguage.googleapis.com/v1beta/openai/ |
| **维度** | 3072 |
| **费用** | 免费额度 |
| **特点** | Google出品，多语言好 |

### 4. Ollama (本地)
| 项目 | 配置 |
|------|------|
| **模型** | nomic-embed-text |
| **Base URL** | http://localhost:11434/v1 |
| **维度** | 768 |
| **费用** | 完全免费 |
| **特点** | 本地运行，隐私好 |
