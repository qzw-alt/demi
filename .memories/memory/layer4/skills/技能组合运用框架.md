## 问题3: 怎么组合运用？

### 5大组合模式

#### 模式A: 深度研究
```
输入: "研究一下XX"
  ↓
multi-search (广度) → tavily (深度) → summarize (总结) → doc-write (记录)
  ↓
输出: 完整研究报告
```

#### 模式B: 内容创作
```
输入: "写篇关于XX的文章"
  ↓
research → outline → write → edit → publish
  ↓
输出: 发布就绪文章
```

#### 模式C: 开发调试
```
输入: "开发XX功能"
  ↓
plan → code → test → deploy → document
  ↓
输出: 完成的功能+文档
```

#### 模式D: 竞品分析
```
输入: "分析一下竞品XX"
  ↓
search → web-fetch → browser-test → analyze → report
  ↓
输出: 竞品分析报告
```

#### 模式E: 决策支持
```
输入: "XX和YY哪个好？"
  ↓
search-options → analyze-pros-cons → recommend
  ↓
输出: 决策建议
```
