## 🎯 研究目标

### 目标1: 记忆压缩算法 (本周)

**问题**: 上下文太长，token 消耗大，加载慢

**研究内容**:
1. **智能摘要技术**
   - 长对话 → 关键决策点
   - 详细过程 → 结果 + 教训
   - 多轮迭代 → 最终方案

2. **分层加载策略**
   - 首屏: 只加载 HOT (活跃任务)
   - 按需: 点击加载 WARM (配置)
   - 搜索: 检索 COLD (历史)

3. **预测性加载**
   - 根据话题预加载相关记忆
   - 根据时间预加载今日待办
   - 根据用户模式预加载偏好

**产出**:
- `scripts/smart-compress.sh` - 智能压缩脚本
- `memory/压缩算法研究.md` - 方法论文档

## 🛠️ 研究方法

### 1. 实验驱动

每个研究目标都通过实际实验验证：
- 假设 → 实验 → 数据 → 结论 → 迭代

### 2. 记录一切

```markdown
## 实验记录模板

### 实验: [名称]
**时间**: 
**假设**: 
**方法**: 
**数据**: 
**结论**: 
**下一步**: 
```

### 3. 快速迭代

- 小步快跑，快速验证
- 失败也记录，避免重复踩坑
- 成功即固化，形成方法论
