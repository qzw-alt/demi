## 5大组合模式

### 1️⃣ 深度研究模式
```
multi-search → tavily → summarize → doc-write
   广度         深度      总结        记录
```
**触发**: 研究、调研、分析
**输出**: 完整研究报告

### 2️⃣ 内容创作模式  
```
research → outline → write → edit → publish
  研究      大纲     写作    编辑    发布
```
**触发**: 写文章、创作、博客
**输出**: 发布就绪文章

### 3️⃣ 开发调试模式
```
plan → code → test → deploy → document
计划    编码    测试    部署     文档
```
**触发**: 开发、实现、添加功能
**输出**: 功能+文档

### 4️⃣ 竞品分析模式
```
search → fetch → browser → analyze → report
搜索     获取     体验       分析      报告
```
**触发**: 分析竞品、对比、研究对手
**输出**: 竞品分析报告

### 5️⃣ 决策支持模式
```
search-options → analyze → recommend
  收集选项        分析       建议
```
**触发**: 怎么选、哪个好、建议
**输出**: 决策建议
