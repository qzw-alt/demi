## 📝 制作说明

### Canva 制作步骤：
1. 打开 canva.com，注册账号
2. 选择 "Presentation" 或 "Social Media" 模板
3. 一帧一页，16:9比例
4. 选深色背景 + 白色大字
5. 关键数字用亮色（红色/橙色）突出
6. 底部加网站域名

### 导出设置：
- 格式：MP4
- 分辨率：1080x1920 (9:16) 或 1920x1080 (16:9)
- 配乐：Canva内置音乐
