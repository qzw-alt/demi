# L签转S2（诊疗）签证办理清单

> 适用场景：外籍人士持L旅游签入境，因病情严重需住院治疗（超10天），需在境内将L签换发为S2（加注"诊疗"）签证并延期

**核心原则：签证类型不影响商保直付，换签仅为合规停留，全程可正常使用国际高端医疗险直付。**

---

## 一、办理前核心准备（3件事）

| 准备事项 | 具体要求 | 目的 |
|---------|---------|------|
| 医院对接 | 联系就诊三甲医院国际部/医务处，说明"L签需转S2诊疗签，需出具官方医疗证明" | 确保医院出具的材料符合出入境审批标准 |
| 商保备案 | 向保险公司申请住院直付预授权，留存预授权号 | 保障住院期间费用直付，无需个人垫付 |
| 住宿登记 | 入境24小时内完成酒店/居住地派出所住宿登记，留存登记证明 | 出入境办理签证的必备材料 |

---

## 二、医院所需材料（核心！出入境审批关键）

### 必须开具的3份官方文件（均需医院公章）

2. **《住院治疗证明》**（原件）
3. **《医疗费用预估单》**（原件）

## 三、出入境管理局办理材料清单

**办理事项：** L签证换发S2（加注"诊疗"）签证+延期  
**办理地点：** 就医城市市级出入境管理局  
**办理时间：** 原L签到期前7–10天（紧急情况可申请加急）

| 材料类别 | 具体材料 | 注意事项 |
|---------|---------|---------|
| 身份证明 | 护照原件+有效L签证页 | 护照有效期≥6个月，空白签证页≥2页 |
| 申请表 | 《外国人签证证件申请表》 | 现场领取填写，或官网提前下载打印 |
| 照片 | 白底2寸免冠证件照1张 | 近6个月拍摄，无刘海、无遮挡 |
| 医疗核心材料 | 医院出具的《诊断书》《住院治疗证明》《费用预估单》 | 全部为原件，必须盖医院公章，缺一不可 |
| 停留合规材料 | 住宿登记证明 | 酒店或派出所出具的正式登记单 |
| 资金保障材料 | 国际商保卡+直付预授权确认单 或 银行存款证明（≥1万美元） | 优先提供商保材料，审批更快 |
| 代办材料（可选） | 委托书+代办人身份证原件+亲属关系证明 | 委托书需患者签字（重病可按指纹代替） |

## 四、办理全流程

### Day 1 - 入院当天
- [ ] 国际部登记：提交护照、L签证、国际商保卡，申请直付预授权
- [ ] 对接医务处：提交《医院证明模板》，说明换签需求

### Day 2-3 - 材料准备
- [ ] 领取医院出具的3份盖章材料
- [ ] 准备好护照、照片、住宿登记证明、商保材料
- [ ] 行动不便者：提前准备好委托书及代办人材料

### Day 4 - 提交申请
- [ ] 前往出入境管理局，取号办理"签证换发+延期"业务
- [ ] 提交全部材料，工作人员审核后出具受理回执

### Day 7-10 - 领取签证（加急可当日/次日）
- [ ] 凭受理回执领取护照，新签证为S2（诊疗）签证
- [ ] 停留期按治疗时长核定（通常≤180天）

---

## 五、关键注意事项

⚠️ **商保直付不受影响**：办理换签期间，患者可正常在医院住院治疗，商保直付流程无需中断

⚠️ **严禁逾期停留**：必须在L签到期前提交换签申请
- 逾期≤10天：每日罚款500元（上限1万元）
- 逾期超10天：可能面临拘留及入境限制

⚠️ **材料真实性**：医院出具的证明必须真实有效，出入境会核查

⚠️ **多科室治疗**：若需转科或转院，需由新医院重新出具医疗证明
