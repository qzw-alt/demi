## 一、网站开发最佳实践

### 1. 页面设计原则
- **移动端优先**：60%流量来自手机，先保证手机体验
- **加载速度**：图片压缩、CSS精简、避免大文件
- **导航一致性**：每个页面都要有完整的导航链接
- **CTA明显**：行动按钮要突出，颜色对比强烈

### 2. 医院页面设计
```
推荐结构：
┌─────────────────┐
│ 搜索框          │
│ [标签筛选区]    │
├─────────────────┤
│ 医院卡片网格    │
│ - 名称          │
│ - 地址          │
│ - 电话          │
│ - [标签1][标签2]│
└─────────────────┘
```

**关键经验**：
- 用标签代替分类，一个医院可以属于多个专科
- JavaScript数组存储数据，方便筛选和扩展
- 添加费用免责声明，避免法律风险

### 3. 博客文章模板
```html
必备元素：
- 吸引人的标题（带数字、利益点）
- 快速回答框（TL;DR）
- 成本对比表
- 步骤流程
- FAQ部分
- CTA转化区块
- 相关文章链接
```

## 二、内容创作技巧

### 1. SEO关键词布局
- 标题包含主关键词
- 前100字出现关键词
- H2/H3标题包含长尾词
- 图片alt标签加关键词
- 结尾相关文章互链

### 2. 高转化内容类型
| 类型 | 转化率 | 例子 |
|------|--------|------|
| 成本对比 | ⭐⭐⭐⭐⭐ | "Save $30,000" |
| 步骤指南 | ⭐⭐⭐⭐ | "7-Step Process" |
| 患者故事 | ⭐⭐⭐⭐ | "Real Experience" |
| 医院排名 | ⭐⭐⭐ | "Top 10 Hospitals" |

### 3. 写作公式
```
标题 = [结果] + [时间] + [目标用户]
例："Giving Birth in China 2026: Save $30,000+ (American Parents Guide)"

开头 = 痛点 + 解决方案 + 利益点
例："US birth costs $30k-50k. In China: $5k-12k. Here's how..."

正文 = 数据 + 案例 + 步骤
结尾 = CTA + 相关链接
```

## 三、工作流程优化

### 1. 批量处理原则
- 一次改完所有页面的导航
- 一次写完所有费用免责声明
- 避免反复切换任务

## 四、与伟烨协作技巧

### 1. 沟通原则
- **主动汇报**：做完立刻说，不要等问
- **选项呈现**：给2-3个方案，让伟烨选
- **进度透明**：还剩多少，预计多久

### 2. 任务管理
```
优先级：
P0 = 今晚必须做（如PayPal电话）
P1 = 明天做（如WhatsApp注册）
P2 = 本周做（如新博客文章）
P3 = 以后做（如数据库化）
```

### 3. 常见需求预判
- "更新页面" = 改数字 + 加链接 + 免责声明
- "写博客" = 成本对比 + 步骤 + FAQ
- "检查网站" = 链接 + 手机端 + 年份
