## 🔧 安装命令参考

```bash

git clone https://github.com/ComposioHQ/awesome-claude-skills.git

# 安装单个技能
cp -r awesome-claude-skills/content-research-writer ~/.config/claude-code/skills/

# 或者使用 skillhub（如果可用）
skillhub install content-research-writer
```
