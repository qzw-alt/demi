## 🔥 黄金组合

### 组合1：深度研究
```
搜索 → 研究 → 写作

web-search/multi-search (广度搜索)
    ↓
tavily-search (深度分析)
    ↓
content-research-writer (整理输出)
    ↓
summarize (摘要)
```

### 组合2：内容创作
```
研究 → 写作 → 优化

multi-search/tavily (找素材)
    ↓
social-content/xhs (写文案)
    ↓
humanizer (优化自然度)
    ↓
summarize (检查要点)
```

### 组合3：客户获取
```
搜索 → 研究 → 记录

lead-research-assistant (找潜在客户)
    ↓
web-search (验证信息)
    ↓
memory-simple (记录到记忆)
```

### 组合4：记忆管理
```
记录 → 分层 → 改进

memory-simple (记录)
    ↓
memory-tiering (分层整理)
    ↓
self-improving-agent (总结改进)
```
