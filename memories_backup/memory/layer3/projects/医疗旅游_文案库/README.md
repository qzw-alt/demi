## ❓ FAQ内容

### 常见问题分类

#### 关于医疗质量
**Q: 中国医院的医疗水平如何？**
A: 中国顶级医院在多个专科领域处于世界领先水平。北京协和、华西医院等常年位居全球医院排名前列。许多中国医生在美国、欧洲接受过培训，医疗设备与国际同步。

**Q: 如何确保手术安全？**
A: 我们合作的均为三甲医院（中国最高等级），拥有JCI国际认证。术前全面评估，术中严格无菌操作，术后24小时监护。

#### 关于费用
**Q: 为什么费用比美国低这么多？**
A: 主要原因：1) 中国医疗人力成本较低；2) 药品和耗材采购价格优势；3) 没有美国高额的医疗保险 overhead。但医疗质量不打折扣。

**Q: 费用包含哪些项目？**
A: 标准套餐包含：手术费、住院费、药品费、检查费、翻译费、机场接送。不包含：往返机票、签证费、家属陪同费用。

#### 关于语言沟通
**Q: 我不会中文怎么办？**
A: 我们提供全程英语/多语言服务：专业医疗翻译陪同就诊，医院国际部有英语护士，术后康复指导提供英文材料。

**Q: 医生会说英语吗？**
A: 合作医院的国际部医生均具备英语沟通能力，部分医生有海外留学背景。

#### 关于签证和行程
**Q: 如何申请中国医疗签证？**
A: 我们提供邀请函和医院预约确认函，您只需向当地中国领事馆申请M字签证（医疗签证），通常3-5个工作日出签。

**Q: 需要在中国停留多久？**
A: 根据手术类型：微创手术3-7天，关节置换2-3周，心脏手术3-4周。我们会制定详细康复计划。
