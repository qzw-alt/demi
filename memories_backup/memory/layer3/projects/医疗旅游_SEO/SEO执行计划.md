## 阶段一: 立即执行 (本周)

### 任务1: 修复首页Title长度
**优先级**: P0 | **预计时间**: 30分钟

**问题**: 当前Title约70字符，可能超出Google显示限制

**行动**:
```html
<!-- 修改前 -->
<title>Medical Tourism China | Cancer, Cardiac & Orthopedic Treatment Guide</title>

<!-- 修改后 -->
<title>Medical Tourism China 2026 | Save 50-80% on Healthcare</title>
```

**验收标准**:
- [ ] Title长度 ≤ 60字符
- [ ] 包含主关键词
- [ ] 包含年份和核心卖点

---

### 任务2: 添加Twitter Card标签
**优先级**: P0 | **预计时间**: 1小时

**问题**: 缺少Twitter Card元数据，分享时显示不佳

**行动**:
为所有主要页面添加:
```html
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="[页面标题]">
<meta name="twitter:description" content="[页面描述]">
<meta name="twitter:image" content="https://chinahospitalsguide.com/images/og-[page].jpg">
```

**验收标准**:
- [ ] 首页有Twitter Card
- [ ] 费用对比页有Twitter Card
- [ ] 博客文章有Twitter Card
- [ ] 通过Twitter Card Validator验证

---

### 任务3: 添加Open Graph标签
**优先级**: P0 | **预计时间**: 1小时

**问题**: 缺少Facebook/LinkedIn分享优化

**行动**:
```html
<meta property="og:title" content="...">
<meta property="og:description" content="...">
<meta property="og:image" content="...">
<meta property="og:url" content="...">
<meta property="og:type" content="website/article">
```

**验收标准**:
- [ ] 所有主页面有完整OG标签
- [ ] 图片尺寸 1200x630px
- [ ] 通过Facebook Sharing Debugger验证

---

### 任务4: 检查并修复robots.txt
**优先级**: P0 | **预计时间**: 30分钟

**问题**: 确保搜索引擎能正确抓取

**行动**:
```
User-agent: *
Allow: /
Sitemap: https://chinahospitalsguide.com/sitemap.xml
```

**验收标准**:
- [ ] robots.txt可访问
- [ ] 没有误屏蔽重要页面
- [ ] 包含Sitemap引用

---

### 任务5: 添加Canonical标签
**优先级**: P0 | **预计时间**: 1小时

**问题**: 防止重复内容问题

**行动**:
为每个页面添加:
```html
<link rel="canonical" href="https://chinahospitalsguide.com/[page-path]">
```

**验收标准**:
- [ ] 所有页面有Canonical标签
- [ ] URL使用绝对路径
- [ ] 指向正确的首选版本
