# 🔑 API Key 简单解释

## 什么是 API Key？

就像**进门的钥匙** 🔑

```
你想使用 OpenAI 的服务
    ↓
OpenAI 说: "请出示钥匙 (API Key)"
    ↓
你提供 API Key
    ↓
OpenAI 让你使用服务
```

---

## 为什么需要 API Key？

### 1. 身份验证
- 证明你是谁
- 防止陌生人使用

### 2. 计费
- 用了多少服务
- 按量收费

### 3. 安全
- 可以随时禁用
- 防止被盗用

---

## 我们的场景

### LanceDB-Pro 需要 API Key 来：

1. **把文字变成向量** (Embedding)
   ```
   "北京协和医院" → [0.1, 0.3, 0.7, ...] (768个数字)
   ```

2. **存储到向量数据库**

3. **搜索时比较相似度**
   ```
   用户问: "北京哪家骨科医院好"
   系统找: 最相似的向量
   返回: "北京协和医院骨科..."
   ```

## 可选方案

### 方案A: OpenAI (最常用)
- **优点**: 效果好，稳定
- **缺点**: 需要付费，需要国外信用卡
- **费用**: $0.0001/1000字符 (很便宜)
- **获取**: https://platform.openai.com

### 方案B: Jina AI (推荐 ⭐)
- **优点**: 免费额度足够，无需信用卡
- **缺点**: 效果略逊于OpenAI
- **费用**: 免费额度 1M tokens
- **获取**: https://jina.ai (注册即得)

### 方案C: Ollama (本地)
- **优点**: 完全免费，本地运行，隐私好
- **缺点**: 需要电脑配置好，速度慢
- **费用**: 0元
- **获取**: 下载安装即可
