## 医院与医生相关

### Q1：阜外医院和协和医院是什么关系？

**简短回答**：两家都是中国顶尖医院，但侧重点不同。阜外医院专注心血管疾病，协和医院是综合性医院。

**详细解答**：
- **阜外医院**：全称"中国医学科学院阜外医院"，是国家心血管病中心所在地，全球最大的心血管疾病诊治中心。在复旦版中国医院专科声誉排行榜中，心血管专科连续13年排名第一。

- **协和医院**：全称"北京协和医院"，是中国医学科学院下属综合性医院，综合实力极强，在复旦版中国医院综合排行榜中连续多年第一。各科室都很强，但心血管专科相对阜外医院规模较小。

**选择建议**：
- 单纯心血管疾病 → 首选阜外医院
- 复杂疑难病症需多科会诊 → 可考虑协和医院
- 需要高端体检+专科诊疗 → 协和医院国际医疗部

### Q2：医生是什么资质？

**简短回答**：主刀医生均为副主任医师以上职称，平均从业经验超过20年。

**详细解答**：
- **职称体系**：中国医生职称分为住院医师、主治医师、副主任医师、主任医师四个等级
- **主刀标准**：心脏外科手术通常由副主任医师及以上职称医生主刀
- **经验水平**：阜外医院年心脏手术量超过4万例，医生年均手术量200-500例
- **学术地位**：多位医生担任国际学术组织委员、核心期刊编委

**如何确认医生**：
- 预约时可要求指定专家级别
- 远程会诊后确定主刀医生
- 重要手术可由主任医师主刀

### Q3：医院有国际认证吗？

**简短回答**：医院均通过中国三级甲等医院评审（最高等级），部分科室参与国际学术交流。

**详细解答**：
- **三甲评审**：中国医院最高等级认证，每4年复审一次
- **国际交流**：阜外医院与克利夫兰诊所、梅奥诊所等有学术交流
- **JCI认证**：部分国际医疗部正在申请JCI认证
- **标准接轨**：诊疗流程、设备标准与国际顶尖医院接轨

## 预约与流程相关

### Q4：如何预约？需要多长时间？

**详细流程**：
1. **初步咨询**（1-3天）
   - 提交病历资料
   - 初步评估可行性
   
2. **远程会诊**（3-5天）
   - 专家评估病情
   - 确定治疗方案
   - 预估费用

3. **确认预约**（1-2周）
   - 支付定金
   - 办理签证邀请函
   - 确定入院日期

4. **行前准备**（1-2周）
   - 办理签证
   - 预订机票酒店
   - 准备病历资料

**总时间**：从首次咨询到入院，通常需要3-6周。

**加急情况**：紧急病例（如主动脉夹层）可启动绿色通道，24-48小时内安排。

**详细解答**：
- **全程翻译**：国际医疗部提供英语、日语等多语种服务
- **陪同翻译**：从接机、就诊到出院，专人陪同翻译

- **日常沟通**：病房护士基础英语交流，紧急情况有翻译随叫随到

**翻译服务范围**：
- 医生问诊
- 检查预约
- 手术知情同意
- 出院指导
- 费用结算

### Q6：家属可以陪同吗？

**详细解答**：
- **签证**：患者和家属均可办理医疗签证（S2签证）
- **陪护**：病房允许一名家属陪护
- **住宿**：
  - 病房陪护床：每晚100-200元
  - 医院周边酒店：300-800元/晚
- **探视**：普通病房探视时间通常为下午2-4点

**建议**：
- 介入手术：1名家属陪同即可
- 开胸手术：建议2名家属轮换陪护

## 治疗与技术相关

### Q7：什么是微创手术？适合我吗？

**详细解答**：

**微创手术类型**：
1. **介入治疗**（经血管）
   - 冠脉支架植入
   - 心脏封堵术
   - TAVR（经导管换瓣）
   - 特点：无切口，恢复快，2-4天出院

2. **胸腔镜/机器人手术**
   - 微创搭桥
   - 微创瓣膜手术
   - 特点：小切口，恢复较快，5-7天出院

**适合人群**：
- 病变相对简单
- 高龄、高危患者
- 希望快速恢复

**不适合情况**：
- 复杂多支病变
- 合并其他心脏问题需同期处理
- 血管入路条件差

**最终方案**：需根据冠脉造影/CT等检查由医生评估确定。

---

### Q8：TAVR手术是什么？

**详细解答**：

**TAVR**（Transcatheter Aortic Valve Replacement）
- 中文：经导管主动脉瓣置换术
- 原理：通过血管将人工瓣膜送至心脏，无需开胸

**手术过程**：
1. 局部麻醉或全身麻醉
2. 导管经股动脉（大腿根部）进入
3. 在X线和超声引导下定位
4. 释放人工瓣膜，替换病变瓣膜
5. 确认位置良好后撤出导管

**优势**：
- 无需开胸、无需体外循环
- 创伤小，恢复快
- 适合高龄、高危患者

**风险**：
- 血管并发症（穿刺处血肿）
- 传导阻滞（可能需要起搏器）
- 瓣周漏

**费用**：25-40万元（含进口瓣膜）

### Q9：手术风险大吗？

**详细解答**：

**阜外医院统计数据**（2024年）：
- 冠脉搭桥手术成功率：98.5%
- 心脏瓣膜手术成功率：97.8%
- 先心病手术成功率：99.2%
- 介入手术成功率：99.5%

**风险因素**：
- 年龄越大，风险越高
- 合并症越多（糖尿病、肾衰等），风险越高
- 急诊手术风险高于择期手术

**降低风险的措施**：
- 术前全面评估
- 术中严密监测
- 术后专业护理
- 并发症及时处理

**医生会在术前详细告知**：
- 手术获益
- 可能风险
- 替代方案
- 拒绝手术的风险

### Q10：术后需要在中国停留多久？

**详细解答**：

**不同手术类型的建议停留时间**：

| 手术类型 | 住院时间 | 建议总停留时间 |
|---------|---------|--------------|
| 冠脉支架 | 2-3天 | 5-7天 |
| 心脏封堵 | 2-3天 | 5-7天 |
| 射频消融 | 2-3天 | 5-7天 |
| TAVR | 3-5天 | 7-10天 |
| 微创瓣膜 | 5-7天 | 10-14天 |
| 开胸搭桥 | 7-10天 | 14-21天 |
| 开胸换瓣 | 7-10天 | 14-21天 |

**飞行建议**：
- 介入术后：1周后可飞行
- 开胸术后：2-3周后可飞行
- 长途飞行（>6小时）：建议术后3-4周

## 费用与保险相关

### Q11：治疗费用大概是多少？

**详细费用参考**（2025年）：

**介入治疗**：
- 冠脉支架（1枚）：6-8万元
- 冠脉支架（2-3枚）：10-15万元
- 先心病封堵：5-10万元
- 房颤消融：8-12万元

**外科手术**：
- 冠脉搭桥（不停跳）：15-20万元
- 冠脉搭桥（体外循环）：18-25万元
- 心脏瓣膜修复：15-20万元
- 心脏瓣膜置换（机械瓣）：18-25万元
- 心脏瓣膜置换（生物瓣）：25-35万元
- TAVR：25-40万元

**复杂手术**：
- 主动脉夹层：30-50万元
- 心脏移植：50-80万元

**费用包含**：
- 住院费、手术费、麻醉费
- 主要耗材（支架/瓣膜）
- 常规检查费、药费

**费用不含**：
- 往返机票
- 酒店住宿
- 餐饮
- 特殊检查（如PET-CT）

## 术后与随访相关

### Q14：回国后如何复查？

**详细解答**：

**复查时间表**：

**介入术后**：
- 1个月：心电图、血常规、凝血功能
- 3个月：心脏超声、运动试验
- 6个月：冠脉CTA（可选）
- 12个月：全面复查

**外科术后**：
- 1个月：胸片、心脏超声、凝血功能
- 3个月：运动试验
- 6个月：全面复查
- 12个月：冠脉CTA或造影

**复查方式**：
1. 在当地医院完成检查
2. 将报告拍照或扫描发送给我们
3. 我们转给主治医生评估
4. 反馈评估结果和建议

**紧急联系**：
- 术后6个月内免费提供咨询
- 紧急情况可通过我们联系主治医生

### Q15：术后需要长期服药吗？

**详细解答**：

**冠脉支架术后**：
- 阿司匹林：长期服用
- 氯吡格雷/替格瑞洛：至少12个月
- 他汀类：长期服用
- 其他：根据病情

**心脏瓣膜术后**：
- 机械瓣：华法林（终身服用，需监测INR）
- 生物瓣：阿司匹林（通常3-6个月）

**搭桥术后**：
- 阿司匹林：长期服用
- 他汀类：长期服用
- 其他：根据病情

**用药注意事项**：
- 严格遵医嘱，不可自行停药
- 定期复查，调整用药
- 注意药物相互作用
- 手术/拔牙前告知医生正在服用抗凝药

### Q16：手术效果能维持多久？

**详细解答**：

**冠脉支架**：
- 药物涂层支架10年通畅率：90-95%
- 可能再狭窄，可再次介入或搭桥

**心脏瓣膜**：
- 机械瓣：可终身使用，但需终身抗凝
- 生物瓣：通常10-15年，之后可能需再次换瓣
- TAVR瓣膜：预期寿命10-15年

**搭桥血管**：
- 动脉桥（乳内动脉）：90%通畅10年以上
- 静脉桥（大隐静脉）：50%通畅10年

**影响因素**：
- 控制危险因素（血压、血糖、血脂、吸烟）
- 规律服药
- 定期复查
- 健康生活方式

### Q18：医疗签证怎么办理？

**详细解答**：

**签证类型**：S2字签证（探亲/医疗）

**所需材料**：
- 护照（有效期6个月以上）
- 签证申请表
- 照片
- 医院邀请函（我们提供）
- 病历资料
- 资金证明

**办理流程**：
1. 确认预约后，我们出具邀请函
2. 向中国驻当地使领馆申请
3. 通常3-5个工作日出签
4. 签证有效期通常为3个月，停留期30-90天

**注意事项**：
- 提前办理，避免延误行程
- 家属可一同申请
- 如需延长停留，可在华申请延期

- VIP病房：套房式，费用最高

**设施**：
- 24小时热水
- 空调/暖气
- 陪护床
