## 冠心病介入治疗案例

### 案例 MY-2025-001

**基本信息**
- 代号：陈先生
- 性别：男
- 年龄：68岁
- 国籍：马来西亚
- 职业：退休商人
- 来源渠道：华人社群推荐

**病情背景**
- 主诉：反复胸痛3年，加重2个月
- 既往史：高血压20年，糖尿病10年
- 当地诊断：三支冠状动脉严重狭窄
- 当地建议：冠状动脉旁路移植术（CABG）

**咨询过程**
- 首次联系：2025年8月15日
- 资料提交：心电图、冠脉CTA、血液检查
- 远程会诊：2025年8月20日（阜外医院心内科主任）
- 方案确定：介入治疗（PCI）

**治疗详情**
- 入院日期：2025年9月10日
- 术前检查：冠脉造影确认病变
  - 左前降支：近段90%狭窄
  - 左回旋支：中段80%狭窄
  - 右冠状动脉：近段85%狭窄
- 手术日期：2025年9月12日、14日（分两次）
- 手术内容：
  - 第一次：前降支植入2枚支架
  - 第二次：回旋支和右冠各植入1枚支架
- 出院日期：2025年9月17日

**康复过程**
- 术后1周：轻微胸闷，逐渐缓解
- 术后1月：可正常行走，无胸痛
- 术后3月：恢复轻度运动（散步、太极）
- 术后6月：复查冠脉CTA，支架通畅

**费用明细**
| 项目 | 金额（人民币） |
|------|--------------|
| 住院费（5天） | 8,000 |
| 冠脉造影 | 6,000 |
| 支架（5枚） | 65,000 |
| 手术费 | 15,000 |
| 药物及检查 | 8,000 |
| 总计 | 102,000 |

**患者反馈**
> "本来马来西亚医生建议开胸搭桥，我很害怕。没想到在中国可以做微创的支架，3天就出院了。医生技术很好，护士也很照顾。现在每天可以散步一小时，感觉像换了个人。"

**经验教训**
- 三支病变并非都必须搭桥，部分患者可选择分期介入
- 糖尿病患者需特别注意术后伤口护理
- 双语病历对后续当地随访很重要

## 先天性心脏病案例

### 案例 ID-2025-003

**基本信息**
- 代号：小丽
- 性别：女
- 年龄：5岁
- 国籍：印度尼西亚
- 家属：父母陪同
- 来源渠道：印尼合作医院转诊

**病情背景**
- 主诉：发现心脏杂音5年，发育迟缓
- 诊断：室间隔缺损（膜周部，8mm）
- 当地建议：开胸修补手术
- 家长顾虑：孩子太小，希望微创治疗

**咨询过程**
- 首次联系：2025年7月通过合作医院
- 资料提交：心脏超声、胸片
- 远程评估：适合经皮封堵术
- 签证办理：医疗签证+父母陪同

**治疗详情**
- 入院日期：2025年8月5日
- 术前检查：
  - 心脏超声确认缺损位置
  - 血常规、凝血功能
  - 胸片、心电图
- 手术日期：2025年8月7日
- 手术内容：经皮室间隔缺损封堵术
  - 导管经股静脉进入
  - 释放10mm封堵器
  - 超声确认位置良好
- 出院日期：2025年8月9日

**康复过程**
- 术后1周：正常活动，无不适
- 术后1月：复查超声，封堵器位置良好
- 术后3月：生长发育追赶，体重增加
- 术后6月：可正常上幼儿园

**费用明细**
| 项目 | 金额（人民币） |
|------|--------------|
| 住院费（4天） | 5,000 |
| 心脏超声 | 800 |
| 封堵器 | 25,000 |
| 手术费 | 12,000 |
| 药物及检查 | 4,000 |
| 总计 | 46,800 |

**患者反馈（家长）**
> "非常感谢中国医生！孩子不用开胸，只是大腿上有个小针眼。手术后第二天就能下床玩了。现在孩子和其他小朋友一样健康，我们全家都很感激。"

**经验教训**
- 小儿先心病介入治疗需严格把握适应症
- 家长心理疏导很重要，需详细解释手术过程
- 术后抗凝治疗需家长配合监测

## 心脏瓣膜病案例

### 案例 US-2025-005

**基本信息**
- 代号：王女士
- 性别：女
- 年龄：72岁
- 国籍：美国（华人）
- 职业：退休教师
- 来源渠道：美国华人社群

**病情背景**
- 主诉：活动后气短2年，晕厥1次
- 既往史：高血压、骨质疏松
- 美国诊断：重度主动脉瓣狭窄
- 美国建议：开胸主动脉瓣置换（SAVR）
- 患者顾虑：高龄，担心开胸风险

**咨询过程**
- 首次联系：2025年6月通过华人社群
- 资料提交：心脏超声、CT、血液检查
- 远程评估：适合TAVR手术
- 决策时间：考虑2周后决定来华

**治疗详情**
- 入院日期：2025年7月15日
- 术前检查：
  - 心脏超声：瓣口面积0.6cm²
  - 冠脉CT：排除严重冠心病
  - 外周血管评估：入路条件良好
- 手术日期：2025年7月18日
- 手术内容：经导管主动脉瓣置换术（TAVR）
  - 经股动脉入路
  - 植入26mm自膨式生物瓣膜
  - 术中超声确认无反流
- 出院日期：2025年7月22日

**康复过程**
- 术后1周：气短明显改善
- 术后1月：可正常行走，无不适
- 术后3月：恢复轻度运动
- 术后6月：复查超声，瓣膜功能良好

**费用明细**
| 项目 | 金额（人民币） |
|------|--------------|
| 住院费（7天） | 15,000 |
| 术前检查 | 8,000 |
| 人工瓣膜 | 180,000 |
| 手术费 | 35,000 |
| 药物及耗材 | 12,000 |
| 总计 | 250,000 |

**患者反馈**
> "美国的医生说必须开胸，我很害怕。在华人群里听说中国有TAVR技术，不用开胸就能换瓣。来了之后发现医院很现代化，医生和护士都很专业。手术第二天就能下床，4天就出院了。现在每天可以散步、买菜，生活质量提高了很多。"

**经验教训**
- TAVR适合高龄、高危患者，但需严格筛选
- 生物瓣膜vs机械瓣膜的选择需充分沟通
- 术后抗凝方案需与患者当地医生衔接

## 心律失常案例

### 案例 TH-2025-007

**基本信息**
- 代号：林先生
- 性别：男
- 年龄：58岁
- 国籍：泰国
- 职业：企业主
- 来源渠道：泰国合作医院转诊

**病情背景**
- 主诉：心悸、胸闷5年
- 诊断：持续性心房颤动
- 既往治疗：药物控制效果不佳
- 泰国建议：射频消融术

**咨询过程**
- 首次联系：2025年5月通过合作医院
- 资料提交：心电图、Holter、心脏超声
- 远程评估：适合射频消融
- 决策：1周后确认行程

**治疗详情**
- 入院日期：2025年6月10日
- 术前检查：
  - 24小时Holter：房颤负荷80%
  - 心脏超声：左房45mm，EF正常
  - 食道超声：排除左房血栓
- 手术日期：2025年6月12日
- 手术内容：三维标测下射频消融术
  - 肺静脉电隔离
  - 左房基质改良
  - 手术时间3小时
- 出院日期：2025年6月14日

**康复过程**
- 术后1周：窦性心律，轻微不适
- 术后1月：无症状，停用抗心律失常药
- 术后3月：复查Holter，无房颤发作
- 术后1年：随访无复发

**费用明细**
| 项目 | 金额（人民币） |
|------|--------------|
| 住院费（4天） | 6,000 |
| 术前检查 | 5,000 |
| 三维标测 | 15,000 |
| 手术费 | 25,000 |
| 药物及耗材 | 8,000 |
| 总计 | 59,000 |

**患者反馈**
> "房颤困扰我很多年，吃药效果不好。在泰国听说中国的射频消融技术很先进，就通过合作医院过来了。手术很顺利，现在一年多了没有复发，也不用再吃那么多药。非常感谢！"

**经验教训**
- 持续性房颤消融成功率略低于阵发性，需提前告知
- 术后3个月空白期内可能复发，需药物维持
- 长期随访很重要，部分患者需二次消融

## 主动脉夹层案例

### 案例 SG-2025-009

**基本信息**
- 代号：张先生
- 性别：男
- 年龄：52岁
- 国籍：新加坡
- 职业：工程师
- 来源渠道：紧急转运

**病情背景**
- 主诉：突发剧烈胸背痛6小时
- 新加坡诊断：Stanford A型主动脉夹层
- 病情紧急：需紧急手术

**紧急处理**
- 首次联系：2025年9月20日（紧急）
- 转运安排：医疗专机转运
- 入院时间：发病后12小时

**治疗详情**
- 入院日期：2025年9月20日（急诊）
- 术前检查：
  - 增强CT：主动脉夹层累及升主动脉
  - 心包积液少量
  - 冠脉开口受累
- 手术日期：2025年9月20日（急诊）
- 手术内容：Bentall手术+主动脉弓置换
  - 主动脉根部置换
  - 主动脉瓣置换（机械瓣）
  - 主动脉弓部分置换
  - 手术时间8小时
- 出院日期：2025年10月5日

**康复过程**
- 术后1周：ICU监护，逐渐稳定
- 术后2周：转普通病房
- 术后1月：可下床活动
- 术后3月：恢复正常生活

**费用明细**
| 项目 | 金额（人民币） |
|------|--------------|
| 医疗转运 | 350,000 |
| 住院费（15天） | 80,000 |
| 手术费 | 120,000 |
| 人工血管及瓣膜 | 150,000 |
| ICU费用 | 100,000 |
| 药物及检查 | 50,000 |
| 总计 | 850,000 |

**患者反馈**
> "当时在新加坡突然发病，医生说必须马上手术，但排队要等。通过紧急转运到了北京，从发病到手术只用了12小时。虽然费用不低，但保住了命。医生技术高超，现在恢复得很好。"

**经验教训**
- A型主动脉夹层是急症，时间就是生命
- 医疗转运需提前规划路线和接收医院
- 术后长期血压管理至关重要
