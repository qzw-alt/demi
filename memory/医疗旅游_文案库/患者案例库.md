## 案例模板

### 案例编号: CASE-001
**患者代号**: 美国心脏病患者-John
**国籍**: 美国
**年龄**: 62岁
**性别**: 男

**病情诊断**:
- 冠状动脉粥样硬化性心脏病
- 三支血管病变（前降支、回旋支、右冠状动脉均>70%狭窄）
- 心功能II级

**选择中国就医原因**:
1. 美国手术等待时间长（约3个月）
2. 费用高昂（预估$150,000+）
3. 了解到阜外医院心血管专科声誉
4. 有中国朋友推荐

**治疗过程**:
| 时间 | 事项 | 详情 |
|-----|-----|-----|
| 第1周 | 远程咨询 | 提交病历资料，获得初步评估 |
| 第2周 | 签证办理 | 医院出具邀请函，申请M字签证 |
| 第3周 | 抵达北京 | 入住医院附近酒店 |
| 第4周 | 术前检查 | 冠脉造影确认病变 |
| 第5周 | 手术 | 微创冠脉搭桥手术（MIDCAB） |
| 第6-7周 | 术后恢复 | ICU 2天，普通病房5天 |
| 第8周 | 出院回国 | 携带康复指导和随访计划 |

**费用明细**:
| 项目 | 费用 (人民币) | 备注 |
|-----|--------------|-----|
| 术前检查 | ¥15,000 | 含冠脉造影 |
| 手术费 | ¥120,000 | 微创搭桥 |
| 住院费 | ¥25,000 | 7天住院 |
| 药品费 | ¥15,000 | 术后用药 |
| 其他 | ¥5,000 | 护理、耗材等 |
| **总计** | **¥180,000** | |

**费用对比**:
- 美国预估费用: $150,000 (约¥1,050,000)
- 中国实际费用: ¥180,000
- **节省: 约83%**

**治疗结果**:
- 手术成功，无并发症
- 术后心绞痛症状完全消失
- 心功能恢复至I级
- 术后3个月复查，恢复良好

**患者反馈**:
> "阜外医院的心外科团队非常专业，术前沟通详细，术后恢复比预期快很多。整个体验超出预期。医生甚至在我回国后还主动跟进我的恢复情况。"

**可用于营销的材料**:

- ✅ 无需漫长等待
- ✅ 术后随访到位

## 案例使用规范

### 隐私保护原则
1. **化名处理**: 所有患者使用化名或代号
2. **信息模糊**: 年龄、日期等做适当模糊处理
3. **授权确认**: 使用前获得患者书面授权
4. **敏感信息**: 不公开具体病历号、身份证号等

### 营销使用建议
1. **匹配受众**: 根据目标患者国籍、病种选择对应案例
2. **真实可信**: 不夸大治疗效果，不虚构数据
3. **多元展示**: 展示不同类型的成功案例
4. **定期更新**: 每季度补充新案例
