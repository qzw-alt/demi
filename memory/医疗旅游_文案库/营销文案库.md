#### 文案2: 深度科普
**标题**: 中国医疗旅游全攻略：从签证到康复

**正文**:
【一、签证办理】
来中国就医，建议申请M字签证（商务签证）。医院可以出具邀请函，说明治疗目的和预计停留时间。通常可以停留30-90天，如需延长可申请延期。

【二、医院选择】
根据病情选择专科强项医院：
- 心血管疾病 → 阜外医院
- 肿瘤治疗 → 医科院肿瘤医院
- 骨科关节 → 积水潭医院、301医院
- 辅助生殖 → 北医三院
- 中医调理 → 广安门医院

【三、费用预算】
除了医疗费用，还要考虑：
- 住宿：医院附近酒店或公寓
- 生活：餐饮、交通
- 翻译：如需专业医疗翻译
- 保险：建议购买医疗旅游保险

【四、术后康复】
术后是否立即回国，需遵医嘱。一般：
- 微创手术：1-2周后可飞行
- 心脏手术：2-4周后
- 骨科手术：2-6周后

【五、远程随访】
回国后，可通过视频或邮件与主治医生保持联系，定期发送复查报告。
