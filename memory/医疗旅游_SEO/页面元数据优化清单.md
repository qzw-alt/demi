## 通用优化检查清单

### 每个页面必须检查:
- [ ] Title长度 50-60字符
- [ ] Meta Description长度 150-160字符
- [ ] 包含核心关键词
- [ ] 独特的title/description (无重复)
- [ ] Canonical URL设置
- [ ] Open Graph标签 (Facebook分享)
- [ ] Twitter Card标签

### 图片优化:
- [ ] 所有图片有描述性alt标签
- [ ] 图片文件名包含关键词
- [ ] 图片尺寸优化 (WebP格式优先)

### 结构化数据:
- [ ] Organization schema (首页)
- [ ] Article schema (博客页)
- [ ] HowTo schema (指南页)
- [ ] FAQ schema (常见问题)
- [ ] BreadcrumbList schema
