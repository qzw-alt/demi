## 🔧 4. 具体优化建议

### 4.1 高优先级 (立即执行)

1. **提交 Sitemap 到 Google Search Console**
   - 创建 sitemap.xml (如不存在)
   - 提交到 GSC
   - 检查收录状态

2. **添加 JSON-LD 结构化数据**
   - Organization schema
   - FAQ schema (费用页)
   - Article schema (博客)

3. **添加 Canonical 标签**
   ```html
   <link rel="canonical" href="https://chinahospitalsguide.com/page/" />
   ```

### 4.2 中优先级 (本周完成)

4. **优化博客内链结构**
   - 相关博客文章互相链接
   - 博客页脚添加"推荐阅读"

5. **增加内容深度**
   - 每个手术类型添加更多常见问题
   - 添加患者视频见证

6. **图片优化**
   - 添加 alt 文本
   - 压缩图片大小
   - 使用 WebP 格式

### 4.3 低优先级 (持续优化)

7. **建立外链**
   - 医疗旅游目录提交

8. **本地SEO**
   - 创建 Google Business Profile
   - 添加更多城市页面
