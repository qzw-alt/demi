### 4.2 现有内容优化

#### 添加FAQ Schema
为每篇博客文章添加结构化FAQ数据，针对问题型关键词:

```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "How much does knee replacement cost in China?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "Knee replacement in China costs $8,000-$12,000, which is 70% less than the US ($35,000-$50,000)."
    }
  }]
}
```
