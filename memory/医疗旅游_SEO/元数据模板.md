## 结构化数据模板

### 医院结构化数据
```json
{
  "@context": "https://schema.org",
  "@type": "Hospital",
  "name": "[医院名称]",
  "description": "[医院简介]",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "[城市]",
    "addressCountry": "CN"
  },
  "medicalSpecialty": ["[科室1]", "[科室2]"],
  "availableService": "International Patient Services"
}
```

### 服务页面结构化数据
```json
{
  "@context": "https://schema.org",
  "@type": "MedicalBusiness",
  "name": "来华就医服务平台",
```

## 使用说明

1. **替换占位符**：将 [方括号] 内容替换为实际信息
2. **字符限制**：
   - Title: 50-60字符（Google显示约600px）
   - Description: 150-160字符
3. **关键词位置**：核心关键词尽量靠前
4. **独特性**：每个页面Title/Description必须唯一
5. **行动号召**：Description中加入CTA（如：Free consultation, Book now）
