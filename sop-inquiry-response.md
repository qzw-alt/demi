# 客户询单响应标准操作流程 (SOP)

> **文档版本**：v1.0  
> **创建日期**：2026-03-10  
> **适用对象**：客服/运营人员

---

## 一、流程概览

```
用户提交询单 → 24小时内响应 → 生成定制攻略 → 发送邮件/WhatsApp → 跟进转化
```

---

## 二、询单信息收集

### 用户填表字段（网站联系表单）

| 字段 | 必填 | 用途 |
|------|------|------|
| 姓名 | ✅ | 邮件称呼 |
| 邮箱 | ✅ | 发送定制攻略 |
| 国家 | ✅ | 签证建议 |
| 病情/治疗需求 | ✅ | 匹配医院 |

| 预计来华时间 | 选填 | 安排预约 |

| 特殊需求 | 选填 | 个性化服务 |

## 三、定制攻略生成流程

### Step 1: 分析用户需求（5分钟）

**提取关键信息：**
- 国籍 → 确定签证类型
- 病情 → 确定科室/医院
- 城市 → 确定推荐医院
- 时间 → 判断紧急程度

**示例：**
> 英国籍 + 种植牙 + 深圳 → 144小时免签或L签 + 北大深圳医院/港大深圳医院

---

### Step 2: 选择攻略模板（2分钟）

| 治疗类型 | 模板文件 | 特点 |
|---------|---------|------|
| 牙科 | `template-dental.md` | 简单治疗，L签即可 |
| 眼科 | `template-eye.md` | 激光手术，1-2天完成 |
| 中医理疗 | `template-tcm.md` | 体验型，旅游结合 |
| 心血管 | `template-cardiology.md` | 需S2签证，住院准备 |
| 肿瘤 | `template-oncology.md` | 长期治疗，复杂流程 |
| 骨科 | `template-orthopedics.md` | 手术+康复 |

---

### Step 3: 填充定制内容（15分钟）

**必须个性化填充的字段：**

```markdown
## 客户信息
- 国籍：[用户国籍]
- 需求：[具体病情]
- 目的地：[城市]
- 生成日期：[日期]

## 一、签证指南
- 推荐签证：[根据国籍和病情推荐]
- 办理流程：[具体步骤]

## 二、医院推荐
- 推荐医院1：[匹配病情的最佳医院]
  - 地址：
  - 电话：
  - 参考价格：[标注仅供参考]
- 推荐医院2：[备选]
- 推荐医院3：[备选]

## 三、治疗流程
- [根据病情描述标准流程]

## 四、参考价格
- [标注"仅供参考，以医院实际报价为准"]

## 五、其他实用信息
- 交通、住宿、保险、中文短语等

## 六、升级服务
- 引导购买Standard套餐
```

---

### Step 4: 审核检查（5分钟）

**检查清单：**
- [ ] 客户信息准确
- [ ] 签证建议符合国籍政策
- [ ] 医院匹配病情和城市
- [ ] 所有价格标注"仅供参考"
- [ ] 包含医院联系方式
- [ ] 有升级服务引导
- [ ] 无错别字

## 五、特殊情况处理

### 情况1: 病情复杂/不确定
**处理：**
- 在攻略中注明"建议先咨询医院确认治疗方案"
- 强调Standard套餐包含"医院预沟通"服务
- 引导用户升级

### 情况2: 用户预算有限
**处理：**
- 推荐性价比高的医院
- 强调Starter Guide已包含足够信息自助就医
- 不强行推销升级

### 情况3: 紧急情况（需尽快治疗）
**处理：**
- 标注加急处理
- 推荐最快可预约的医院
- 主动提供电话/视频咨询

### 情况4: 用户国家签证政策特殊
**处理：**
- 查询最新签证政策
- 必要时建议用户咨询当地中国使领馆
- 提供备选方案（如第三国中转）

## 八、常见问题 FAQ

**Q1: 用户没有明确意向城市怎么办？**
A: 根据病情推荐最合适的城市（如心血管→北京，肿瘤→广州/上海），并提供2-3个选项。

**Q2: 用户病情我们不熟悉怎么办？**
A: 先查询该病情中国哪家医院最强，再生成攻略。必要时标注"建议先咨询医院确认治疗方案"。

**Q3: 用户询单信息不完整怎么办？**
A: 邮件回复询问补充信息，同时提供基于现有信息的初步建议。

**Q4: 用户收到攻略后没有回复怎么办？**
A: 按跟进时间线执行，Day 3和Day 7主动跟进，但不要过于频繁打扰。
