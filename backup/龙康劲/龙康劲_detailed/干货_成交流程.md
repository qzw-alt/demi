# 外贸成交流程拆解

## 主题
外贸询盘转化难？看看成交漏斗

## 成交漏斗五步走

### 📥 第一步：询盘接收（10%）
- 客户发来询盘
- 关键：2小时内回复
- 话术：确认需求 + 体现专业 + 邀约下一步

### 📋 第二步：需求确认（8%）
- 详细了解客户具体需求
- 关键：问清楚规格、数量、交期、目标价
- 工具：需求确认清单

### 💼 第三步：方案报价（5%）
- 提供专业方案和报价
- 关键：报价单要专业、完整、有吸引力
- 注意：体现价值，而非单纯比价

### 🤝 第四步：谈判跟进（3%）
- 处理客户疑问、讨价还价
- 关键：价值塑造、差异化竞争
- 技巧：案例背书、样品策略

### ✅ 第五步：成交付款（1%）
- 合同确认、收款发货
- 关键：付款方式要明确、风险要可控

## 成交真相

- 100个询盘 → 10个报价 → 3个谈判 → 1个成交
- **重点**：把漏斗每一层都做好，结果自然好
